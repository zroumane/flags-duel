# flags-duel

A duel game where you have to find the country associated with a flag. You can play it on [Flags Duel](https://flags-duel.zph.ovh).

## Usage
```
yarn install
```

### Development
```
# server side (Express.js)
yarn dev

# client side (Vue.js + Router)
yarn watch
```

### Production
```
yarn start
```
or you can use docker
